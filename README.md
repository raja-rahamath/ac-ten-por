# ac-ten-por

Tenant Customer Portal for service request management.

## Overview

Customer-facing portal for:
- Viewing service requests
- Tracking appointment status
- Checking invoice history
- Managing profile settings

## Port

- **3001** (Development)

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Setup

```bash
pnpm install
cp .env.example .env
pnpm dev
```
